# 软件 & 利器

  • 开发人员使用的软件版本 应该 尽可能保持一致
  • 如果没有软件版本的要求，应该 优先选用软件的稳定版

# 一些推荐使用的软件

  • 代码编辑器
    • VSCode
    • WebStorm
  • 浏览器
    • Chrome
    • Firefox
  • 接口调试工具
    • Postman
    • Postwoman
  • Git GUI
    • Fork
  • SSH
    • Xshell: 国外版官网可以下载免费版本，支持同时打开 4 个标签页
    • Putty
  • FTP
    • Xftp: 国外版官网可以下载免费版本，支持同时打开 2 个标签页
    • WinSCP
  • 集成开发环境
    • Laragon
  • 解压缩工具
    • 7z
  • 攥写
    • Bear: 仅支持 Mac 和 iOS 的高颜值 Markdown 笔记软件
    • Typora: 所见即所得的高颜值 Markdown 编辑器
    • 印象笔记
  • 邮箱
    • Outlook 桌面版
    • Spark: 支持 Mac, iOS 和 Android 的邮箱软件
  • 云盘
    • 坚果云
    • Dropbox
  • 其它生产力工具
    • 石墨文档: 在线文档
    • 滴答清单: GTD 工具
    • 1Password: 密码管理工具
    • 图灵社区: 正版 IT 电子书
    • ColorHunt: 好看的配色合辑
